# DAY-5: PySpark Transformations and Type Casting
# Objective

On Day 5, I explored the PySpark environment using the Databricks Community Edition and gained hands-on experience with PySpark transformations and type casting techniques.

# Key Learnings
Introduction to the PySpark workspace and home screen in Databricks
Importing and working with CSV files in PySpark
Reading data into DataFrames
Performing column-level operations using:
withColumn() for creating or modifying columns
Column replacement and transformation
Converting data types (e.g., string to integer) using type casting
Dropping unnecessary columns from DataFrames
Basic data cleaning and transformation techniques
# Summary

This session provided a solid foundation in handling and transforming data using PySpark. I learned how to efficiently load, manipulate, and clean datasets, which are essential skills for data engineering workflows.
